# Wondrous Places Beta

Experimental releases for the Wondrous Places Dalamud plugin.

Wondrous Places brings venue-focused roleplay prompts and interactions into an in-game Dalamud window. The Covenstead is its first venue module.

## Install

1. Open Dalamud Settings in Final Fantasy XIV.
2. Open the Experimental tab.
3. Add this URL under Custom Plugin Repositories:

   `https://raw.githubusercontent.com/Sydsquicious/wondrous-places-beta/main/repo.json`

4. Save your settings.
5. Open the Plugin Installer and install **Wondrous Places**.

This is an experimental custom-repository release and may contain bugs. Learn more about the Covenstead at [covenstead.carrd.co](https://covenstead.carrd.co).

Wondrous Places was written by Braely Fiala.
